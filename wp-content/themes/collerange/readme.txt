CollerAnge has been built on Top of Coller WordPress theme. To use this theme you should have Coller(http://wordpress.org/themes/coller/) Installed.

++Differences from Parent Theme++

The Major Changes are in Color Combinations. The Accent has been changed to Orange. The Primary Navigation has been re-designed. The Post Area and the Sidebar have been assigned a Fixed Color. 

Borders have been added to various sections to give them a distinct look. Font Family has been changed.
